#### Endless Runner
Developed in response to a university assignment to create an endless runner using the tutors own 2D engine, ASGE. 

#### ASGE Foundations
ASGE is built upon the following technologies
1. C++11
2. GLEW
3. GLFW
4. GLM
5. STB
6. PDCurses
