Faster than Darkness

A Unity game jam based off the theme "Faster than...". 

html5 templates

This project was a 1 week game jam based off the theme "Faster than...". The game is a parkour based endless runner where the player tries to escape the darkness chasing them. The game focuses on parkour based first person movement and quick reactions. The player has the ability to slide, wall run and grapple allowing for some challenging obstacles.

My role during this project was to create the first person character controller and all extra movements which came with it, as well as making a few obstacles for the course. I really enjoyed this as it was my first time creating a first person controller. I am happy with how it turned out and would like to expand on the character controller in the future to turn it into a more advanced movement system.
