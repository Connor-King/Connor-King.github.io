Dragon Mockery (In Beta)

A Unity game jam based off the theme "Shooter game where you suffer mockery of dragons in beta". 

This project was a fun game jam game based off the theme "Shooter game where you suffer the mockery of dragons in beta". The game is a rogue-like dungeon crawler where you explore a procedurally generated dungeon killing enemies on your way to the dragon boss which uses insults as attacks.

This was a fun project to work on as a game jam as the limited time (1 week) and the random theme made for an interesting combo. My role during this game jam was to create the enemies AI, attacks and the animation controllers to go along with them.
